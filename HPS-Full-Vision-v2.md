## Architecture pivot (locked)

### Full Map spine
```
Birth → stage → stage → … → Now
                              ↓
                    Present positioning
                    (skills, resources, patterns, plan, kill list)
```

Not seven equal parallel journeys as primary nav.

### Impact Lite (~25–35 min)
Same spine, compressed:
- 3 chapters only
- Bright memories (up to 5) before optional dark door
- Named people each chapter
- Learned sentence + still-alive habit
- Present: hate-pattern, substitute behavior, kill list, 7-day act, connection +/−

### Pre-session mindset gate
Cinematic slides before questions:
1. Life as forest/garden — beauty + scar-plants
2. Who you are now = years lived; slightly lighter shoulders
3. No sugar, no court — self with self
4. Memory is powerful and unreliable; photos/family help; skip dark OK
5. Timeline then present; Lite or Full

Optional soft ambient pad (Web Audio). Skip intro always available.

## Ontology (compressed)

Human = learning machine forced by time to choose; pain/pleasure loop; memory grades and updates self-model.

```
Behavior = incentives_now + old_pain + learned_scripts + available_options
         + imagined_identity + social_audience + fear + desire + biology
```

Load-bearing constructs: **Script · Mask · Status game · Shame vs guilt · Agency continuum · Proof**

Canonical axes: Survival, Day, Body, People, Worlds, Past stages, Shadow (dual harm), Scripts, Resources, Agency, Future, Proof.

Culture = OS. MENA/Morocco physics matter (honor, family mesh, papers, village visibility). Western therapy language is not neutral ground truth.

## Methods

### Question order (locked)
What happened → who → where/when → what you did → felt → learned → shows today → what we do (L4 only).

### Depth levels L1–L4
Raw → compare/intensity → meaning/effect → continuity/plan. Never open L3/L4 before L1 (or explicit skip).

### Causal engine
Link types: installs, triggers, constrains, rewards, punishes, models, protects, costs, competes, enables.  
No single-root-trauma dogma. Competing explanations required for high-drama claims.

### Stage card complete-enough
place + people + one good memory + learned sentence + still-running behavior + auto-summary template.

### Person card law
Write the other as a full human (day, money, fear, pressure) — not only “what they did to me.”
